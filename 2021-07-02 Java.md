'a' = 문자

"a" = 문자열 ("a\0")



char c = 'a' ;

c라는 char 데이터 타입에 a가 대입



String str1 = "Hello" ;

어딘가에 존재하는 Hello라는 상수(리터럴)의 위치를 str1에 대입

위치(주소)가 어딘지 알고있을뿐 값 자체가 str1에 대입되는게 아님.

문자열 : 시작위치 ~ \0 까지 (시작위치는 str1이 알고있음)
