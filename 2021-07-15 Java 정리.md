클래스 배열

* 배열을 만들어서 객체를 넣어줘야 한다.
* int[ ] b = new int[3];    // 배열 
  * int 객체생성 하는 애가 아니다 그냥 바로 쓸 수 있다.
* A[ ] a = new A[3];        // 배열 객체
